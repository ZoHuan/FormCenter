## 三步。就三步。

| | 你做什么 | 系统做什么 |
|--|---------|-----------|
| ① | 拖拽组件搭表单 | 自动适配 PC 和手机 |
| ② | 点复制发链接 | 填写者打开就能填 |
| ③ | 点导出 | Excel 下载到本地 |

不注册。不付费。不求技术。

---

## 给谁用的

HR 收简历的、行政做登记的、销售搞调研的、老师发问卷的。

**一句话：任何需要"让别人填个表"的场景。**

## 凭什么选它

| 😰 你的担心 | 😎 我们的回答 |
|------------|-------------|
| 要学吗？ | 不用。拖就完了。 |
| 要装啥？ | 浏览器。就一个浏览器。 |
| 数据安全吗？ | 全在你电脑里，不经过任何服务器。 |
| 收费吗？ | 免费，MIT 开源。今天免费，明天也免费。 |
| 手机能用吗？ | 填写页专门为手机优化。扫码就能填。 |
| 断网了？ | 照样用。离线数据在电脑里好好的。 |

## 30 种组件

输入框、多行文本、数字、单选、多选、下拉、级联、日期、图片上传、手写签名、评分、表格、地图定位、二维码……

> 请假申请 · 活动报名 · 满意度调查 · 客户信息登记 · 设备巡检 · 投票表决
>
> 一个工具，所有场景。

---

## 跑起来

```bash
npm install && npm run dev
```

浏览器打开 `http://localhost:3000`，动手拖拽。
